# 🧧 合成年夜饭 🧧

一个有趣的合成游戏，类似"合成大西瓜"，点击投放食物，相同的食物碰撞会合成更大的年夜饭！

## 🎮 玩法

- 点击游戏区域上方投放食物
- 相同的食物碰撞会自动合成更大的食物
- 从饺子🥟开始，最终目标是合成满汉全席🥡
- 当食物堆积到顶部时游戏结束

## 🍽️ 合成链

```
饺子 → 春卷 → 红烧肉 → 清蒸鱼 → 大虾 → 螃蟹 → 火锅 → 龙虾 → 烤鸭 → 佛跳墙 → 满汉全席
```
