# 🧧 合成年夜饭 🧧

一个有趣的合成游戏，类似"合成大西瓜"，点击投放食物，相同的食物碰撞会合成更大的年夜饭！

## 🎮 玩法

- 点击游戏区域上方投放食物
- 相同的食物碰撞会自动合成更大的食物
- 从饺子开始，最终目标是合成佛跳墙
- 当食物堆积到顶部（超过虚线）时游戏结束
- 合理利用颠勺道具调整布局

## 🥄 颠勺道具

- 每局游戏可使用 **1 次**
- 点击颠勺按钮，将所有食物向上抛起
- 帮助调整食物布局，避免过早游戏结束
- 使用后会有 2 秒冷却期

## 🍽️ 合成链

```
饺子 → 汤圆 → 春卷 → 清蒸鲈鱼 → 东坡肉 → 八宝饭 → 白切鸡 → 大闸蟹 → 清蒸龙虾 → 鲍鱼捞饭 → 佛跳墙
```

## 🎯 游戏规则

1. 点击游戏区域上方任意位置投放食物
2. 相同等级的食物碰撞会自动合成更高级的食物
3. 合成时会获得分数，食物越大分数越高
4. 当 3 个或更多食物稳定在顶部虚线以上时，游戏结束
5. 合理使用颠勺道具可以延长游戏时间
